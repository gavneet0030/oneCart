  </h2>

  <p style="
      font-size: 1.05em;
      color: #2c3e50;
      max-width: 950px;
  ">
    <strong>OneCart</strong> is a comprehensive full-stack online e-commerce web
    application that enables users to browse products, manage a shopping cart,
    place orders, and track purchase history through a secure and interactive
    user interface. The system is designed to replicate the core functionality
    of real-world e-commerce platforms while maintaining simplicity,
    modularity, and scalability.
  </p>

  <p style="
      font-size: 1.05em;
      color: #2c3e50;
      max-width: 950px;
      margin-top: 12px;
  ">
    The application is divided into three major components:
    a <strong>frontend user interface</strong>,
    a <strong>backend API server</strong>, and
    a <strong>database layer</strong>.
    Each component is developed independently but communicates seamlessly
    through RESTful APIs, ensuring clear separation of concerns and easier
    maintenance.
  </p>
